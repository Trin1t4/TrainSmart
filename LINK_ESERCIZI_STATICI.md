## NOTE TECNICHE

### Formato consigliato per l'app:
- **Illustrazioni:** SVG o JPG 1080x720 (Spotebi)
- **Foto:** JPG/WebP max 1200px larghezza
- **Compressione:** Usare TinyPNG/Squoosh per ottimizzare

### Per ogni esercizio statico serve:
1. Una immagine della posizione corretta
2. (Opzionale) GIF animata per transizioni
3. Descrizione testuale dettagliata (gia' in exerciseDescriptions.ts)

### Tutte le immagini sopra sono:
- Gratuite per uso commerciale
- Non richiedono attribuzione (eccetto dove specificato)
- Alta risoluzione
