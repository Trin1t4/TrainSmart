# Exercise Image Generation Prompts

Questa guida contiene i prompts per generare immagini degli esercizi mancanti usando Leonardo AI, ChatGPT (DALL-E) e Perplexity.

**Totale esercizi senza video: 50**

---

## TEMPLATE BASE

### Leonardo AI
```
Professional fitness photography, gym environment, athletic person demonstrating [EXERCISE NAME], side view showing proper form and technique, clean background, high contrast lighting, sharp focus on muscle engagement, 4K quality, realistic proportions, anatomically correct posture
```

### ChatGPT (DALL-E)
```
Create a professional fitness illustration showing the correct form for [EXERCISE NAME]. The image should show an athletic person in workout clothes performing the exercise with proper technique. Clean gym background, good lighting, emphasis on body positioning and form. High quality, realistic style.
```

### Perplexity (Playground/Imagen)
```
High-quality fitness photography: [EXERCISE NAME] exercise demonstration. Athletic model in gym attire showing proper execution form. Professional studio lighting, clean background, focus on body mechanics and posture. Realistic, detailed, suitable for fitness app.
```
